# React TypeScript App

React + TypeScript + Vite で構築された開発環境です。

## ミドルウェア

- Node 20.19
- Typescript 4.3.5

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## プロジェクト構成

- `src/` - ソースコード
- `public/` - 静的ファイル（必要に応じて作成）
- `dist/` - ビルド成果物（自動生成）

## 開発サーバー

開発サーバーを起動すると、通常は `http://localhost:5173` でアクセスできます。
